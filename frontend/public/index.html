UBC Course/Department Search Page

Please enter the department[1] you wish to search for or the course [2] you wish to search and the results will be displayed below.

_____________________________________________

[1] = All departments must be valid.
Note that a valid department includes the department's complete short name or a substring of the shortname. (E.g "cpsc" or "ps" are both valid inputs)
If you choose to input a substring, You will recieve results for any department that includes that substring.
(E.g: input = "th" -> results = "math...., anth.... (etc)")
Note that a substring of a department and a course number (E.g: input = "ps 310") also works!

[2] = All courses must be valid.
Note that a valid course includes the department's complete short name or a substring of the shortname (see [1]) and course number or a substring of course number (E.g "cpsc 110" or "ps 11" are both valid inputs).
If you choose to input a substring of department shortname and substring of course number, the course number substring length must be at least 2 numbers.

_____________________________________________
